# Poster

The poster is a community site where you can share your Thoughts & Opinions about topics with others.

## Routers

### Home

- /
  => Home Page
- /login
  => Login Page
- /join
  => Join Page
- /search?q=a
  => Search Page

### Posters

- /posters/new
  => Create new Poster Page

- /posters/:postername
  => Watch Poster

- /posters/:postername/edit
  => Edit Poster

### Topics

- /topics/new
  => Create new Topic Page

- /topics/random
  => Watch Random Topic Page

- /topics/:topicname
  => Watch Topic

- /topics/:topicname/edit
  => Edit Topic

### User

- /:username
  => Watch User

- /:username/edit
  => Edit User
